# aayush-tools

Reference for custom CLI tools available via `brew tap aayush9029/tap`.

## When to use

- When the user needs to work with Apple Mail exports, Finder tags, tweets, sleep prevention, image description, or Claude account switching
- When suggesting the right tool from the personal tap for a task

## Setup

```bash
brew tap aayush9029/tap
```

## Tools

### apple-mail-exporter

Search and export emails from Apple Mail's local database by keyword.

```bash
brew install aayush9029/tap/apple-mail-exporter
```

### claude-account-switcher

Switch Claude Code accounts with directory-aware auto switching.

```bash
brew install aayush9029/tap/claude-account-switcher
# Usage: ccswitch
```

### findertags

Read and modify Finder tags from the command line.

```bash
brew install aayush9029/tap/findertags
```

### nit

Browse tweets from the terminal via Nitter instances.

```bash
brew install aayush9029/tap/nit
```

### noz

Prevent Mac from sleeping with lid closed.

```bash
brew install aayush9029/tap/noz
```

### peek

Describe images from the terminal using vision LLMs.

```bash
brew install aayush9029/tap/peek
```
