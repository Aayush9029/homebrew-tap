# noz

Prevent Mac from sleeping with the lid closed.

## Install

```bash
brew install aayush9029/tap/noz
```

## Options

| Flag | Description | Default |
|------|-------------|---------|
| `-m, --minutes N` | Duration in minutes | `45` |
