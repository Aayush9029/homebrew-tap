# claude-account-switcher

Switch between Claude Code accounts, with optional directory-based auto switching.

## Install

```bash
brew install aayush9029/tap/claude-account-switcher
```

## Usage

```bash
ccswitch
```

Run `ccswitch --help` for all options.
