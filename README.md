# Homebrew Tap

Custom Homebrew formulae by [Aayush Pokharel](https://github.com/Aayush9029).

## Install

```bash
brew tap Aayush9029/tap
```

## Formulae

| Formula | Description |
|---------|-------------|
| `apple-mail-exporter` | Search and export emails from Apple Mail's local database by keyword |

### apple-mail-exporter

```bash
brew install apple-mail-exporter
```
